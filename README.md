# PowerElectronics
Circuit simulations &amp; PCB designs

This repo contains the circuits simulations and PCB designs I have done while learning general electronics and power electronics.

## PCBs
These have been designed using KiCAD 9 and KiCAD 10.
Just open the project files in KiCAD to view the schematics and pcb layout. You can use the HQ DFM software (Windows only) or its plugin in KiCAD if interested in Design for Manufucturing.

## Electronics Circuits Simulations
Being a heavy user of GNU/Linux, opensource and free software I have used QUCS-S [https://github.com/ra3xdh/qucs_s] with Ngspice [https://ngspice.sourceforge.io/] as the simulation kernel. Qucs with SPICE offers a nice GUI for instruments of  schematic capture, visualization and provides different passive and active components including a very rich device library that is updated regularly.
