## IRFZ44 Switch Test
This example shows how realistic switching circuits 
could be simulated with Ngspice and Xyce on QUCS-S. This 
approach can be used for simulation of power electronics 
in time domain. Also this illustrates usage of library 
components with Ngspice and Xyce.
