## Full-wave Diode Bridge Rectifier
It's simple, uses 4 diodes for rectification, meaning it's uncontrolled. Uses one Capacitor for filtering the resulting dc signal.
A more robust uncontrolled rectifier incooporates better filtering to try as much to minimize the undesired ac component from the incomplete attenuation of the harmonics os the low pass filter.
